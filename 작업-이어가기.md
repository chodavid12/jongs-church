## 편집 가능화 패턴 (Phase 4 반복작업)
1) 요소에 data-cms="키"(글)/data-cms-src(이미지·영상)/data-cms-href(링크). 정적값은 폴백 유지.
   링크: <a data-cms-href="k_link"><span data-cms="k_label">문구</span></a>
2) site_settings 시드(멱등): insert (key,published_value,draft_value,label,group_name,sort) values (...,'현재값','현재값','한글라벨','<페이지>',N) on conflict(key) do nothing.
   group_name = 페이지명(home/about/worship/life/news).
3) admin 폼 없으니 GROUP_META 등 안 건드려도 됨. HTML 변경은 배포, DB 시드는 즉시.
